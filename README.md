🔥 Smart Fire Hazard Detection System

An Arduino-based IoT project developed during the IoT Assistant Internship Training Program at IIIT Kottayam.

📖 Overview

The Smart Fire Hazard Detection System is designed to detect fire hazards at an early stage using sensors and provide immediate alerts. When a potential fire is detected, the system activates a buzzer and LED to notify nearby people, helping reduce the risk of accidents and property damage.

This project demonstrates the use of Arduino and IoT concepts to build a simple, reliable, and cost-effective fire safety system.

---

✨ Features

- 🔥 Detects fire using a flame sensor
- 🌡️ Monitors temperature
- 🚨 Activates buzzer during fire detection
- 💡 Turns ON LED as a visual warning
- ⚡ Fast response to hazardous conditions
- 💰 Low-cost and easy to implement

---

🔌 Circuit Connections

Component| Arduino Pin
Flame Sensor| D2
Temperature Sensor| A0
Buzzer| D8
LED| D13
VCC| 5V
GND| GND

---

⚙️ Working Principle

1. The flame sensor continuously checks for the presence of fire.
2. The temperature sensor monitors the surrounding temperature.
3. If fire or abnormal temperature is detected, the Arduino processes the sensor data.
4. The buzzer sounds an alarm immediately.
5. The LED turns ON to provide a visual warning.
6. Once the hazard is removed, the system returns to monitoring mode.
